## Technologie
- HTML5
- CSS3 (Bootstrap)

## Funkcjonalności
- responsywny layout (Bootstrap)
- system nawigacji z kategoriami
- sekcje newsów i podgląd artykułów
- carousel (Owl Carousel) dla wyróżnionych treści
- podstrony (kategorie, kontakt)
- funkcjonalność wyszukiwania
- uporządkowana struktura komponentów
## Rozszerzenie o logowanie

## W ramach dodatkowego przedmiotu (Bezpieczeństwo danych / aplikacje webowe) projekt został rozszerzony o:

- system logowania użytkowników (PHP)
- integrację z bazą danych MySQL
- uwierzytelnianie oparte na sesjach
- podstawową walidację danych wejściowych
